    • About Oregon Trail MMO

    This is the final project for CS4241 - Webware designed by Thomas Schweich, Alex Hard, and Dan Wensley. Oregon trail is a classic game irst designed in 1971, with numerous iterations made in later years. This version is designed off of the original text based 1971 version, with some modern elements added from later games. Where this version differs is through its interface and multiplayer opportunities. This website has been designed to replicate the look and feel of a computer the first Oregon Trail was made for, the xerox sigma 9. This computer used a rounded squre monitor that was vector based rather than pizel based. We have attempted to replicate that through font and CSS choices. Further this version allows players to interact and trade with each other, provided they are in the same location.
